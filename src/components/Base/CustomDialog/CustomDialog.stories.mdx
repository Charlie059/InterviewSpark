# CustomDialog

The `CustomDialog` is a highly reusable component that displays a dialog with a title and message. It includes buttons for confirming or canceling the action and can be easily customized according to the needs of the interface.

## Usage

Use this component wherever you need to get confirmation from the user or inform them about important information.

- `open`: Indicates whether the dialog is open or not.
- `onClose`: Callback function invoked when the dialog needs to be closed.
- `title`: Sets the title of the dialog.
- `content`: Defines the content inside the dialog.
- `actions`: The actions displayed at the bottom of the dialog.
- `sx`: The sx props for the dialog.
