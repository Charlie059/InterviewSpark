# FrontEnd Page, Components and Layout Proposal

Components as minimum basic elements of one Page, we
have BaseUI component which is the wrappers or modifications of base Material UI components,
and AssembledUI which is combined or layout-related components based on basicUI and Material UI components.

Data are fetched at the Page level and distribute via Context,
since multiple components under this page might require the same data
